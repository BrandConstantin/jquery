Para que jQuery funcione en tu web tienes que ir a jQuery.com y descargar la libreria, luego incluir esta libreria en tu web. Se hace a través de etiqueta script incluyda en la cabecera:


		<script src="jquery-3.1.1.min.js"></script>

Para que jquery funcione, el documento tiene que empezar con la siguiente linea:


		$(document).ready(function(){
			//aqui los métodos de jQuery
		});

Veamos algunos ejemplos con selectores:


		//basandose en el nombre del elemento 'p'
		$(document).ready(function(){
			$("p").click(function(){
				$(this).hide();
			});
		});

		//basandose en el #id
		$(document).ready(function(){
			$("button").click(function(){
				$(#test).hide();
			});
		});
		.........
		<p id="test">Click aqui para eliminar texto</p>

Si se emplea el simbolo all (*) se esconderán todos los elementos.

Para esconder un elemento se puede emplear uno de los siguientes modos:

  • $("p.ejemplo").hide();
  • $("p:first").hide();
  • $("p:first").hide();
  • $("ul li:first").hide();
  • $("[href]").hide();
  • $("a[target = '_blank']").hide();
  • $("a[target != '_blank']").hide();
  • $(":button").hide(); - para esconder un boton
  • $("tr:even").css("background-color", "yellow"); - para establecer un color de background a la filas impares
  • $("tr:odd").hide(); - para establecer un color de background a la filas pares

En jQuery hay una seria de eventos:

  • click - cuando se hace click sobre un elemento
  • dbclick - cuando se hace doble click sobre un elemento
    • 
				$(document).ready(function(){
				    $("p").dblclick(function(){
				        $(this).hide();
				    });
				});
  • mouseenter - cuando el ratón pasa por encima de un elemento:
    • 
				$(document).ready(function(){
				    $("#alerta").mouseenter(function(){
				        alert("Has pasado por encima!");
				    });
				});
  • mouseleave - cuando el ratón sale del área de un elemento:
    • 
				$(document).ready(function(){
				    $("#alerta").mouseleave(function(){
				        alert("Estas abandonando el area!");
				    });
				});
  • mousedown - cuando el boton del ratón esta presionado hacia izquierda, derecha o en medio:
    • 
				$(document).ready(function(){
				    $("#principal").mousedown(function(){
				        alert("Alerta!");
				    });
				});
  • mouseup - lo mismo que mousedown pero la función se ejecuta no cuando se presiona el ratón, sino cuando se deja de ser presionado:
    • 
				$(document).ready(function(){
				    $("#principal").mousedown(function(){
				        alert("Alerta!");
				    });
				});
  • hover - es una combinación de mouseenter() y mouseleave():
    • 
				$(document).ready(function(){
				    $("#principal").hover(function(){
				        alert("Has entrado en el cuadraito!");
				    },
				    function(){
				        alert("Ahora, te has salido del cuadraito!");
				    });
				});
  • focus - se ejecuta cuando un campo de un formulario se esta utilizando:
    • 
				$(document).ready(function(){
				    $("input").focus(function(){
				        $(this).css("background-color", "#123");
				    });
				    $("input").blur(function(){
				        $(this).css("background-color", "#789");
				    });
				});
  • on - se utiliza para agrupar más de un método:
    • 
				$(document).ready(function(){
				    $("p").on({
				        mouseenter: function(){
				            $(this).css("background-color", "yellow");
				        },
				        mouseleave: function(){
				            $(this).css("background-color", "lightgreen");
				        },
				        click: function(){
				            $(this).css("background-color", "orange");
				        }
				    });
				});

Efectos

jQuery dispone de varios efectos, con cual se puede hacer un monton de cosas interesantes. Veamos:

  • hide() - esconder
  • show() - mostrar
    • 
				$(document).ready(function(){
				    $("#esconder").click(function(){
				        $("p").hide(2000);
				    });
				    $("#mostrar").click(function(){
				        $("p").show(4000);
				    });
				});
  • toggle() - hace lo mismo que hide() y show():
    • 
				$(document).ready(function(){
				    $("button").click(function(){
				        $("#principal").toggle();
				    });
				});
  • fade() - que hace los elementos visibles. Hay 4 tipos de métodos fade:
    • fadeIn() - muestra elementos ocultos y tiene como valores 'slow', 'fast' y segundos:
      • 
						$(document).ready(function(){
						    $("button").click(function(){
						        $("#d1").fadeIn();
						        $("#d2").fadeIn("slow");
						        $("#d3").fadeIn(3000);
						    });
						});
    • fadeOut() - oculta elementos visibles y tiene los mismos valores que fadeIn():
      • 
						$(document).ready(function(){
						    $("button").click(function(){
						        $("#d1").fadeOut();
						        $("#d2").fadeOut("slow");
						        $("#d3").fadeOut(3000);
						    });
						});
    • fadeToggle() - junta las funciones de fadeIn() y fadeOut():
      • 
						$(document).ready(function(){
						    $("button").click(function(){
						        $("#d1").fadeToggle();
						        $("#d2").fadeToggle("slow");
						        $("#d3").fadeToggle(3000);
						    });
						});
    • fadeTo() - se usá para dar opacidad a los elementos:
      • 
						$(document).ready(function(){
						    $("button").click(function(){
						        $("#d1").fadeTo("slow", 0.1);
						        $("#d2").fadeTo("slow", 0.5);
						        $("#d3").fadeTo("slow", 0.9);
						    });
						});
  • sliding() - que mueve los elementos hacia arriba o hacia abajo, y se divide en:
    • slideDown() - que desliza los elementos hacia abajo:
      • 
						$(document).ready(function(){
						    $("#d1").click(function(){
						        $("#d2").slideDown("slow");
						    });
						});
    • slideUp() - desliza un elemento hacia arriba:
      • 
						$(document).ready(function(){
						    $("#d1").click(function(){
						        $("#d2").slideUp("slow");
						    });
						});
    • slideToggle() - desliza un elemento tanto hacia arriba como hacia abajo:
      • 
						$(document).ready(function(){
						    $("#d1").click(function(){
						        $("#d2").slideToggle("slow");
						    });
						});
  • animate() - se usá para crear animaciones:
    • 
				$(document).ready(function(){
				    $("button").click(function(){
				        var div = $("#d1");
				        div.animate({height: '600px', opacity: '0.5'}, "slow");
				        div.animate({width: '600px', opacity: '0.9'}, "3000");
				        div.animate({height: '100px', opacity: '0.5'}, "slow");
				        div.animate({width: '100px', opacity: '0.9'}, "3000");
				    });
				});
  • stop() - este método se utiliza para parar las animaciones ante de que este acaben de por si:
    • 
				$(document).ready(function(){
				    $("#stop").click(function(){
				        $("#d2").stop();
				    });
				});
  • callback() - se utiliza para hacer una llamada a una función, esta función se ejecuta una vez que el efecto anterior ha acabado 100%. También existe la posibilidad de que primero hace la llamada a una función y luego se ejecuta la anterior:
    • 
				$(document).ready(function(){
				    $("button").click(function(){
				        $("p").hide("slow", function(){
				            alert("Parágrafo escondido!");
				        });
				    });
				});

				$(document).ready(function(){
				    $("button").click(function(){
				        $("p").hide(2000);
				        alert("Parágrafo escondido!");
				    });
				});
  • también se pueden usár varias acciones/métodos al mismo tiempo:
    • 
				$(document).ready(function(){
				    $("button").click(function(){
				        $("#p1").css("color", "red")
				            .slideUp(2000)
				            .slideDown(2000);
				    });
				});

jQuery Html

jQuery tiene unos métodos muy poderosos para cambiar y manipular elementos y atributos HTML. Tiene los siguientes métodos para manipular DOM:

  • text() - establece o devuelve el contenido tipo text de los elementos selectados
  • html() - establece o devuelve el contenido de los elementos selectados incluyendo las etiquetas HTML
    • 
			$(document).ready(function(){
			    $("#btn1").click(function(){
			        alert("Text: " + $("#test").text());
			    });
			    $("#btn2").click(function(){
			        alert("HTML: " + $("#test").html());
			    });
			});
  • val() - establece o devuelve el valor de un campo
    • 
			    $("btn").click(function(){
			        alert("Valor: " + $("#test2").val());
			    });
  • attr() - se utiliza para obtener el valor de un atributo
    • 
				    $("btn").click(function(){
				        alert($("#jQuery-gratis").attr("href"));
				    });

text(), html() y val() se pueden utilizar para establecer contenido de siguiente modo:


		    $("#test4Boton1").click(function(){
		        $("#test4").text("Parágrafo 1 cambiado");
		    });
		    $("#test5Boton2").click(function(){
		        $("#test5").html("Parágrafo 2 cambiado!");
		    });
		    $("#test6Boton3").click(function(){
		        $("#test6").val("COSTY");
		    });

Los 3 métodos mencionados anteriormente tienen una función de que se pueden volver a llamar. La función de rellamamento tiene 2 parámetros, una de index, valor nuevo, que es el elemento actual seleccionado y el valor antiguo:


		    $("#test7btn1").click(function(){
		        $("#test7").text(function(i, origText){
		            return "Texto antiguo: " + origText + " Nuevo texto: Hola alumno! (index: " + i + ")";
		        });
		    });

Para establecer un valor diferente a attr() se hace de este modo:


		    $("button").click(function(){
		        $("#test9").attr("href", "http://www.google.com");
		    });

Añadir elementos

Con jQuery es muy facil añadir elementos. Para esto se usá uno de los siguientes métodos:

  • append() - inserta contenido al final de los elementos selectados
    • 
			    $("#btn1").click(function(){
			        $("ol").append("
  • Item
    • ");
			    });
  • prepend() - inserta contenido al principio de los elementos selectados
    • 
			    $("#btn2").click(function(){
			        $("ul").prepend("
  • Item
    • ");
			    });
  • after() - inserta contenido despues de los elementos selectados
  • before() - inserta contenido ante de los elementos selectados
    • 
				$(document).ready(function(){
				    $("#btn3").click(function(){
				        $("img").before("Ante");
				    });

				    $("#btn4").click(function(){
				        $("img").after("Despues");
				    });
				});

    También se pueden añadir varios elementos al mismo tiempo:

    
			    function despues() {
				    var txt1 = "Te ";                   // crear elemento con html
				    var txt2 = $("").text("amo ");    // crear elemento con jQuery

				    txt3.innerHTML = " mi amor!
    ";
				    $("#amor").after(txt1, txt2, txt3);      // insertar elemento despues de img
				}

Eliminar elementos

Para eliminar elementos se utiliza:

  • remove() - elimina los elementos y sus hijos
  • empty() - elimina los elementos
    • 
			    //test 6
			    $("button").click(function(){
			        $("#eliminar").remove();
			    });

			    //test 7
			    $("button").click(function(){
			        $("#vaciar").empty();
			    });

Manipular CSS

Con jQuery se conocen varios métodos para manipular CSS:

  • addClass() - añade una o más clases a un elemento selectado
    • 
			    $("#btn1").click(function(){
			        $("h4, p").addClass("azul");
			    });
  • removeClass() - elimina una o más clases de un elemento selectado
    • 
			    $("#btn2").click(function(){
			        $("h4, p").removeClass("azul importante");
			    });
  • toggleClass() - alterna entre eliminar y añadir clases de un elemento
    • 
			    $("#btn3").click(function(){
			        $("#p1, #p2, #p3").toggleClass("azul importante");
			    });
  • css() - establece o devuelve el estilo de un atributo
    • 
			    $("button").click(function(){
			        alert("El color de background es = " + $("p").css("background-color"));
			    });

		        $("#btn5").click(function(){
			        $(".b1, .b2").css({"background-color": "yellow", "font-size": "200%"});
			    });

Trabajar con las dimensiones

jQuery ofrece posibilidad de trabajar con varios métodos en relación a las dimensiones:

  • width()
  • height()
    • 
			    $("#btn1").click(function(){
			        var txt = "";
			        txt += "Tamaño div: " + $("#d1").width() + "
    ";
			        txt += "Altura div: " + $("#d1").height();
			        $("#d1").html(txt);
			    });
  • innerWidth() - devuelve el tamaño del elemento incluido el padding
  • innerHeight() - devuelve la altura del elemento incluiendo el padding
  • outerWidth() - devuelve el tamaño del elemento incluiendo el padding y border
  • outerHeight() - devuelve el altura del elemento incluiendo el paddin y el border
  • para devolver tamaño/altura de un elemento inclyendo padding, border y margin se incluie la palabra true entre parantesis:
    • 
			    $("#btn2").click(function(){
			        var txt = "";
			        txt += "Tamaño div: " + $("#d2").width() + "
    ";
			        txt += "Altura div: " + $("#d2").height() + "
    ";
			        txt += "Outer width del div (margin incluido): " + $("#d2").outerWidth(true) + "
    ";
			        txt += "Outer height del div (margin incluido): " + $("#d2").outerHeight(true);
			        $("#d2").html(txt);
			    });
  • redimensionar un div
    • 
			    $("#btn3").click(function(){
			        $("#d3").width(600).height(600);
			    });

Acendientes y Descendientes

En jQuery se puede recorrer el árbol DOM para encontrar los antepasados de un elemento. Hay tres formas de hacerlo:

  • parent() - devuelve el padre directo de un elemento
    • 
				$(document).ready(function(){
				    $("span").parent().css({"color": "red", "border": "2px solid red"});
				});
  • parents() - devuelve todos los 'antepasados' de un elemento hasta el elemento html
    • 
				$(document).ready(function(){
				    $("span").parents().css({"color": "red", "border": "2px solid red"});
				});
  • parentsUntil() - devuelve el padre de un elemento y el padre de este
    • 
				$(document).ready(function(){
				    $("span").parentsUntil("div").css({"color": "red", "border": "2px solid red"});
				});

Para recorrer el árbol DOM con sus decendientes tenemos:

  • children() - devuelve todos los hijos directos de un elemento seleccionado, o seleccionando un hijo en especial utilizando un parámetro para la busqueda:
    • 
				$(document).ready(function(){
				    $("div").children().css({"color": "red", "border": "2px solid red"});
				});

				$(document).ready(function(){
				    $("div").children("p.first").css({"color": "red", "border": "2px solid red"});
				});
  • find() - devuelve los elementos descendientes de un elemento seleccionado, hasta el ultimo:
    • 
				$(document).ready(function(){
				    $("div").find("span").css({"color": "red", "border": "2px solid red"});
				});

				$(document).ready(function(){
				    $("div").find("*").css({"color": "red", "border": "2px solid red"});
				});

Paso (traspaso) entre hermano

En jQuery es algo normal saltar de un elemento a otro en el árbol DOM. Esto se hace con:

  • siblings() - devuelve todos los hermanos de un elemento
    • 
				//todo lo que tiene alrededor
		    	$("h2").siblings().css({"color": "red", "border": "2px solid red"});

		    	//los hermanos que son p
	    	    $("h2").siblings("p").css({"color": "red", "border": "2px solid red"});
  • next() - devuelve el siguiente elemento hermano
    • 
				//todo lo que tiene alrededor
		    	$("h2").next().css({"color": "red", "border": "2px solid red"});
  • nextAll() - devuelve todos los hermanos de un elemento seleccionado
  • nextUntil() - devuelve todos los hemanos que hay entre dos argumentos especificados
    • 
    			$("h2").nextUntil("h6").css({"color": "red", "border": "2px solid red"});
  • prev() - devuelve el elemento anterior
  • prevAll() - devuelve todos los elementos anteriores
  • prevUntil() - devuelve los elementos que hay entre dos argumentos especificados

Filtración de elementos

Los métodos más basicos para la filtración son:

  • first() - devuelve el primer elemento que hay despues del elemento seleccionado
    • $("div p").first().css("background-color", "lightblue");
  • last() - devuelve el ultimo elemento que hay
    • $("div p").last().css("background-color", "lightgreen");
  • eq() - devuelve el elemento especificado con el número de index
  • filter() - devuelve los elementos que coinciden con un criterio
    • $("p").filter(".diferente").css("font-size", "12px");
  • not() - devuelve los elementos que no coinciden con un criterio
    • $("p").not(".diferente").css("font-size", "12px");
